# IoMT Network Security Monitoring on Raspberry Pi

Ce dépôt contient le rapport final du projet portant sur un système IoT de monitoring de la sécurité réseau dans un environnement hospitalier IoMT.

## Contenu

- `Rapport_IoT_final.pdf` : rapport final du projet

## Description

Le projet traite :
- la détection d’intrusions dans le trafic réseau IoMT
- l’utilisation de modèles de Machine Learning légers
- le déploiement sur Raspberry Pi
- la mesure du temps, CPU, RAM et énergie logicielle
- la visualisation des résultats dans un dashboard local
